# Dossier Images

Placez vos images dans ce dossier.

## Images requises

### Photo de profil
- **Nom du fichier** : `profile.jpg` ou `profile.png`
- **Taille recommandée** : 500x500px minimum
- **Format** : JPG ou PNG
- **Utilisation** : Photo de profil dans la section Hero

### Favicon
- **Nom du fichier** : `favicon.png`
- **Taille** : 32x32px ou 64x64px
- **Format** : PNG
- **Utilisation** : Icône du site dans l'onglet du navigateur

### Images de projets
- **Noms des fichiers** : `project1.jpg`, `project2.jpg`, `project3.jpg`, `project4.jpg`
- **Taille recommandée** : 800x600px minimum
- **Format** : JPG ou PNG
- **Utilisation** : Captures d'écran de vos projets dans la section Projets

## Conseils

- Compressez vos images pour améliorer les performances
- Utilisez des noms de fichiers descriptifs
- Optimisez la qualité vs taille de fichier
