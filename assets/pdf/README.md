# Dossier PDF

Placez votre CV en format PDF dans ce dossier.

## Fichier requis

### CV PDF
- **Nom du fichier** : `cv.pdf`
- **Utilisation** : Téléchargement du CV via le bouton dans la section Hero
- **Format** : PDF
- **Taille recommandée** : < 2 MB

## Conseils

- Utilisez un nom de fichier cohérent : `cv.pdf` ou `CV_VotreNom.pdf`
- Assurez-vous que le PDF est optimisé (pas trop lourd)
- Vérifiez que le contenu est à jour avant de le publier
- Si vous changez le nom du fichier, mettez à jour le lien dans `index.html`

## Modification du lien dans HTML

Si vous utilisez un nom différent de `cv.pdf`, modifiez cette ligne dans `index.html` :

```html
<a href="assets/pdf/cv.pdf" download class="btn btn-secondary">
```

Remplacez `cv.pdf` par le nom de votre fichier.
